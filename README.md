# HG3X8-4V8

Minimalist Weather Dashboard — une application météo légère et responsive.

## Concept

Un tableau de bord météo minimaliste qui affiche les conditions actuelles et les prévisions à 5 jours. Interface épurée, chargement rapide.

## Fonctionnalités

- Géolocalisation automatique
- Recherche par ville
- Conditions actuelles (température, humidité, vent)
- Prévisions sur 5 jours
- Mode clair / sombre

## Technologies

- HTML5, CSS3, JavaScript vanilla
- Aucun framework, aucune dépendance

## Déploiement

```bash
git clone https://github.com/C8dn01kw72mw/HG3X8-4V8.git
cd HG3X8-4V8
# Ouvrir index.html dans le navigateur
```
